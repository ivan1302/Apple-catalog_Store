# Apple Catalog.Store

This project is a responsive React + TypeScript product catalog inspired by Apple device collections. The website displays phones, tablets, and accessories with a modern UI, interactive product cards, product detail pages, favourites, and a shopping cart experience.

# Live Demo

Experience the live website: [Apple-catalog Store DEMO LINK](https://ivan1302.github.io/Apple-catalog_Store/)

# Getting Started

- git clone https://github.com/ivan1302/Apple-catalog_Store.git
- cd Apple-catalog_Store
- npm install
- npm start

# Features

- Responsive catalog layout for phones, tablets, and accessories
- Home page with featured products, category previews, and slider components
- Product listing pages with sorting, pagination, and item count controls
- Detailed product pages with image galleries, technical specs, and related products
- Favourites list saved in localStorage
- Shopping cart saved in localStorage with checkout modal
- Mobile-friendly hamburger menu and responsive navigation
- Loading animation with Lottie
- Client-side routes using React Router DOM

# Notes

The app uses a local JSON API under `public/api`, so it runs directly in a browser environment without an external backend.
